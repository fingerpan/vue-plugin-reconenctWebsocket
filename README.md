# vue-sockets
websocket plugin for Vue2.JS

## Installation
``` bash
# npm install vue-socket --save
```

### Development Setup

``` bash
# install deps
npm install

# build dist files
npm run build

# serve examples at localhost:8080
npm run dev

```

## start
``` javascript
 import Socket from 'vue-socket'
 import Vue from 'vue'
 // new
 let socket = new Socket(url)
 // 使用插件
 Vue.use(Socket, socket)

```
